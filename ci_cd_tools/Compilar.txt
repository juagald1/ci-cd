//Ejemplo de compilar por CMD
// 1 Declarar workspace (debe estar libre, cerrar code composer si lo esta usando)
// 2 Si code composer ha abierto el proyecto almenos 1 vez lo añade a la carpeta .metadata del workspace, por esta razon solo hace falta la ruta del workspace
// 3 Añadir nombre del proyecto (-ccs.projects)
// 4 La compilacion la realiza buscando la carpeta Debug del proyecto (-ccs.configuration)

// nota: permite compilar un proyecto, por tanto si el proyecto tiene mas de una cpu abra que ejecutar el comando dos veces (una por cpu)

eclipsec -noSplash -data "C:\Users\Usuario\workspace" -application com.ti.ccstudio.apps.projectBuild -ccs.projects test_28027 -ccs.configuration Debug
